h1.  NTS for NodeJS

将nts移植到nodeJS上运行

依赖<code>jsdom</code>,<code>htmlparser</code>,<code>express</code>,<code>connect</code>,第一次安装请执行

<pre class="console">
npm install jsdom
npm install htmlparser
npm install express
npm install connect
</pre>

h2.  通过NPM安装 nodejs-nts

需要预装 npm

<pre class="console">npm install nts</pre>

h2.  通过源码安装 nodejs-nts

需要预装 npm 和 git

<pre class="console">
git clone git://github.com/nomospace/nodejs-nts.git
cd nodejs-nts/
npm install .
</pre>

h2. 打印随机数

新建文件test.js

<pre class="console">
	var NTS = require('.nts');
	console.log(NTS.U._$randNumberString(10));	
</pre>

运行

<pre class="console">node test.js</pre>
